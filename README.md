## Features

- [x] Basic UI using ConstraintLayout
- [x] Search for a movie
- [x] Display results in a list

## Stack used

- Butter Knife `8.5.1`
- Jackson `2.9.2`
- FastAdapter `3.0.4`
- Android `8.0`
- Java `8`

## Getting started

#### Requirements

- JDK `8`
- Android Studio `3`

#### Installation

```sh
git clone https://github.com/jclerc/learning-android.git
cd learning-android
```

Then import project into Android Studio and you're done.
